# @loopkit/core

Event-sourced agent plane kernel — an append-only ledger, a single fold, a doctor/self-heal
step, an SLO board, and a CLI (`loopctl`) for driving a two-beat (reactor + dispatch) autonomous
build loop.

Status: experimental, pre-1.0 (v0.1).

See `../../docs/event-model.md` for the target-model direction this package is heading toward,
and `prompts/` for the (skeleton, customize-per-setup) LLM prompts the reactor/dispatch beats use.
